1. Initializing¶

Cosmos is initialized using the cosmos.Cosmos object, which is subsequently used to starts instances of Execution which contain the entire structure of a workflow.

from cosmos import Cosmos
cosmos = Cosmos('sqlite:///my_cosmos_db.sqlite')
cosmos.initdb()
execution = cosmos.start('My_Workflow')

You will likely have to install the python package required to use the driver you’d like. For example, if you are using postgres+psycopg2:///, make sure you pip install psycopg2. See SQLAlchemy Engines for more details.

1.1.1. Cosmos¶

__init__(database_url='sqlite:///:memory:', get_submit_args=<function default_get_submit_args>, default_drm='local', default_queue=None, flask_app=None)[source]¶
Parameters:
  • database_url (str) – A sqlalchemy database url. ex: sqlite:///home/user/sqlite.db or mysql://user:pass@localhost/database_name or postgresql+psycopg2://user:pass@localhost/database_name
  • get_submit_args (func) – a function that returns arguments to be passed to the job submitter, like resource requirements or the queue to submit to. See cosmos.default_get_submit_args() for details
  • flask_app (Flask) – A Flask application instance for the web interface. The default behavior is to create one.
  • default_drm (str) – The Default DRM to use (ex ‘local’, ‘lsf’, or ‘ge’)
initdb()[source]¶

Initialize the database via sql CREATE statements. If the tables already exists, nothing will happen.

resetdb()[source]¶

Resets (deletes then initializes) the database. This is not reversible!

runweb(host, port, debug=True)[source]¶

Starts the web dashboard

cosmos.default_get_submit_args(task, default_queue=None)[source]¶

Default method for determining the extra arguments to pass to the DRM. For example, returning “-n 3” if task.drm == “lsf” would caused all jobs to be submitted with bsub -n 3.

Parameters:
  • task (cosmos.Task) – The Task being submitted.
  • default_queue – The default queue.
Return type:

str
