7.1. Tool¶

class cosmos.models.Tool.Tool(tags)[source]¶

Essentially a factory that produces Tasks. cmd() must be overridden unless it is a NOOP task.

__init__(tags)[source]¶
Parameters:tags – (dict) A dictionary of tags.
cmd(inputs, outputs, **kwargs)[source]¶

Constructs the command string. Lines will be .strip()ed.

Parameters:
  • inputs – (list) Input TaskFiles, each element corresponds to an abstract_input_file in self.inputs
  • outputs – (list) Output TaskFiles, each element corresponds to an abstract_output_file in self.outputs
  • kwargs – (dict) Parameters passed via tags.
Returns:

(str) The text to write into the shell script that gets executed
